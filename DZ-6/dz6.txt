1. Показать информацию по потокам. В отчет вывести номер потока, название курса и дату начала занятий.
2. Найти общее количество учеников для каждого курса. В отчет вывести название курса и количество учеников по всем потокам курса.
3. Найти среднюю оценку по всем потокам для всех учителей. В отчет вывести идентификатор, фамилию и имя учителя, среднюю оценку по всем проведенным потокам. Учителя, у которых не было потов, также должны попасть в выборку.



1. 
SELECT
training_groups.number AS 'stream_number',
courses.name AS 'course_name',
training_groups.started_at
FROM training_groups INNER JOIN courses
ON training_groups.course_id = courses.id;

2.
SELECT 
courses.name AS 'course_name',
SUM(students_amount) AS 'students_amount'
FROM training_groups INNER JOIN courses
ON training_groups.course_id = courses.id
GROUP BY courses.name;

3.
SELECT
teachers.id,
teachers.surname,
teachers.name,
AVG(performance) AS 'average grade'
FROM teachers LEFT JOIN academic_performance
ON teachers.id = academic_performance.teacher_id
GROUP BY teachers.id;
